Project Name:       To find the median of the input array.

Project Descrition: We take elements of the array by the user.
                    We use function Find_median to find the median. 
                    The program is hardcoded with 100 as the capacity of the array.
                 
Installation:       Compile the project using command:gcc -o median median.c
                    Run the project by using command:./median

Usage:              This program can be used to find the median of the input array.
